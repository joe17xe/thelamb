## 2. Arguments retenus / écartés
- **Retenus** : antériorité matérielle (Qumrân, Septante) ; densité manuscrite ; variantes cataloguées publiquement ; témoignage exposé (noms, lieux, 1 Co 15) ; le renvoi coranique aux Écritures en circulation (5:47 ; 10:94) — présenté factuellement, sans triomphalisme.
- **Écartés** : toute statistique invérifiable ; tout argument « par le nombre de conversions » ; toute comparaison moqueuse avec d'autres corpus religieux.
- **Cadre d'honnêteté (affiché sur la page)** : les manuscrits prouvent la transmission, pas l'inspiration ; croire reste une réponse libre. → C'est notre différenciateur de crédibilité.
